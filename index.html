Pathbuilder JSONs

For each sheet, go to Menu > Export JSON > View JSON. Copy all the text there into the respective text areas. The page autosaves the last inputted data to your browser's local storage in case you navigate away from the page.

Make sure you copy one JSON in before doing the next. Pathbuilder only stores one json per user, and overrides the old one when you hit Export JSON.
